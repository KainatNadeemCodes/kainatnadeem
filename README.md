# Kainat Nadeem — Portfolio

Personal portfolio website of **Kainat Nadeem** — AI Engineer, Frontend Developer, UX Designer, and Social Impact Technologist. Built to support MSc AI applications and STEM scholarship opportunities.

🔗 **Live:** [kainat-nadeem.vercel.app](https://kainatnadeem.vercel.app)

---

## About

This portfolio was designed to communicate one thing clearly:

> *"She builds intelligent systems with purpose."*

It covers my work across AI engineering, full-stack development, inclusive UX design, and social impact technology — including projects like an AI-powered health assistant, NextGenShe (a women's STEM initiative), and Haven (a neuro-inclusive safety platform).

| Layer | Technology |

## Getting Started

Clone and run locally:

```bash
# Clone the repo
git clone https://github.com/KainatNadeemCodes/portfolio.git

# Navigate into the project
cd portfolio

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Deployment

This site is deployed on **Vercel**. Every push to `main` triggers an automatic redeployment.

To deploy your own fork:

1. Push the repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Import Project
3. Select the repo → Vercel auto-detects Vite settings
4. Click Deploy
